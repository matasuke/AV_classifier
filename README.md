# AV classifier

## Description
classification network for av actresses.

## Requirements
+ Python >= 3.3
   - Keras >= 1.2
   - openCV >= 3.1
   - dlib >= 19.4
   - tqdm >= 4.14

## Results
- Final Result
    + train loss: 0.4556
    + train accuracy: 94.4%
    + validation loss: 1.224
    + validation accuracy: 72.2%
